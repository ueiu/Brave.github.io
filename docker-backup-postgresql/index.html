什么是Docker Volume？

在操作之前，我们要先了解一下Docker Volume ，这样才方便理解我们之后的操作。
Docker Volume：翻译为数据卷，用于持久化数据以及容器之间共享数据，他挂载于宿主机上。所以即使删掉了某个容器，数据依然会存在我们宿主机中，保证数据的安全性。
